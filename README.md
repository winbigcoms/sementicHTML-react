# 시멘틱한 html과 css를 위한 ediya 클론 프로젝트

## 목표
- 웹 개발의 기본이 되는 html과 css를 목적에 맞고 ui의 흐름과 역할에 맞게 작성한다.
- 웹 접근성과 웹 표준을 검사한다. 
- 반응형 웹으로 제작한다.

- SPA이지만 SEO에 대해 생각해본다.
